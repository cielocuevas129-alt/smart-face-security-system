Smart Face Security System
Sistema de detección facial en tiempo real desarrollado con Python y OpenCV.
Descripción
Este proyecto utiliza visión por computador para detectar rostros mediante una cámara web. El sistema identifica rostros en tiempo real, muestra información en pantalla y almacena evidencias automáticamente.
Funcionalidades
Detección de rostros en tiempo real.
Conteo automático de personas detectadas.
Visualización de fecha y hora en pantalla.
Captura automática de imágenes cuando se detectan rostros.
Registro de eventos en archivo CSV.
Interfaz de monitoreo mediante cámara web.
Tecnologías utilizadas
Python
OpenCV
CSV
Datetime
